# Uttalehjelp

- **Gracias** → *grathias* (i Madrid) eller *grasias* (andre steder)
- **Por favor** → por favor
- **¿Cuánto cuesta?** → kuanto kuesta
- **La cuenta** → la kuenta
- **¿Dónde está...?** → donde esta
- **Agua sin gas** → agua sin gas
- **Cerveza** → therbetha (Madrid) eller servesa
